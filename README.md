### PicGo图床插件和图床服务端简单介绍

**介绍**

这是一款需要配合个人自定义图床使用的PicGo插件。

**展示**

![image-20210323170242902](http://47.98.234.89/images/image-20210323170242902.png)

**picgo-plugin-tmage**

1. 内容简单，适合想要开发个人图床的同学参考学习。 
2. 需要配合PicGo同时使用。

**tmage**

1. 图片上传的服务端。
2.  使用spring boot开发。 
3. 需要自己部署到服务器上。

**总结**

1. 适合喜欢折腾的同学学习。 
2. 开发完会对服务器知识，文件上传知识，nginx，tomcat等内容有一定的了解。
